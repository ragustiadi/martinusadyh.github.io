Column Selection Mode in NetBeans IDE

Dec 2nd, 2012

Karena dikomporin teman yang menunjukkan dukungan Column Selection di text editor Sublime, akhirnya iseng-iseng apakah NetBeans IDE mempunyai fitur yang sama dan ternyata ada loh :) Untuk mengaktifkan Column Mode Selection di NetBeans, tekanlah kombinasi tombol CTRL+SHIFT+R atau tekanlah tombol dengan gambar di source mode. Dan dibawah ini adalah tampilan Column Selection di NetBeans IDE.

Fitur ini baru saya coba di NetBeans IDE v7.2 :) Jadi untuk versi dibawah ini, saya belum tahu :D

Ubuntu Global Menu Support in Swing Application

Aug 25th, 2012

Setelah minggu kemarin kita mencoba plugin Java Swing Ayatana di NetBeans, sekarang mari kita coba bagaimana menggunakan library Java Swing Ayatana pada aplikasi kita :) Sekarang buatlah dahulu sebuah Java Project biasa pada NetBeans IDE, kemudian buatlah sebuah JFrame Form dan tambahkan JMenuBar pada JFrame tersebut dan design hingga tampak seperti gambar dibawah ini : [caption id=“attachment_1957” align=“alignnone” width=“300”] Contoh Design JFrame dan JMenuBar[/caption]

Jika sudah, sekarang download-lah dahulu library terbaru Java Swing Ayatana disini (versi terbaru pada saat tulisan ini dibuat adalah jayatana-1.2.3.jar) kemudian tambahkan pada project kita hingga tampilan project kita seperti gambar dibawah ini : [caption id=“attachment_1958” align=“alignnone” width=“215”]Library Java Swing Ayatana Pada Project[/caption]
